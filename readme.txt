Current Author's Posts - Display posts of current author only.

== Description ==

* Users with 'admin' role will be able to view posts of all users.
* Users other than 'admin' role will be able to view own posts only, posts of other users will be unavailable to him/her.

== Installation ==

* Download and install plugin from wordpress.org , Activate it
* After activation of this plugin registered users who are not admin will be able to see only their added posts at wordpress admin.
